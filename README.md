# Tower Defense Rush
OOP Semester Project — FAST NUCES
Built in C++ using SFML

## How to Compile and Run
1. Install SFML 2.x from https://www.sfml-dev.org/download.php
2. Link SFML in Visual Studio project settings
3. Compile all .cpp files and run
